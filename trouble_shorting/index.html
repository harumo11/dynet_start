トラブルシューティング

nvidia-smiがうまく表示されない．apt installを勧められる．

前日まで問題なく使用できていたということは，前日行ったapt upgradeによって nvidia関連のライブラリが上書きされてしまった可能性があります． この場合は，プロプライエタリなライブラリをもう一度インストールしましょう．

sudo apt install cuda
